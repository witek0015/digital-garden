## Lecture notes:
- **Statistics** - show only *one vector at a time* (one variable analyses)
- **Chemometrics** - give maximum relevant chemical information
- There are two main chemometrics **machine learning methods** - *supervised* and *unsupervised*
- **Unsupervised machine learning**, in which *we don't know the data*, seeks patterns simultaneously, from the whole dataset and allows for graphing based on all of the data (HCA, PCA)
- **Supervised machine learning** is when we have some *additional vector axis*, i.e. we take a part of the data set as learning set, and then make predictions and calculations based on the regression made with learning set
- **Data mining** - a practice that extracts the data that we need
	- reducing dimensionality
	- taking biggest impacting factors
	- visualization
	- classification
- **Data preparation** - in order to mine the data and use it in a proper manner, we need to do some things first:
	- remove duplicates
	- remove outliers
	- handle missing data
	- consistent data formatting
	- ***normalizing data***
	- documenting and versioning
