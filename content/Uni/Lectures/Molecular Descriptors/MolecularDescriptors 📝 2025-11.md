## Lecture notes:
- Molecular descriptors can be defined as mathematical representations of molecules' properties that are generated by algorithms.
- Tools for descriptors:
	- alvaDesk
	- cok descriptor GUI
	- BlueDesc
	- ChemAxon
	- MOLD2
	- Toxicity Estimation Software Tool
	- Open3DQSAR
	- Dragon
	- PaDEL-descriptor
	- ADMEWORKS modelbuilder
	- QuBiLS-MIDAS
- Intrinsic properties (molecular volume, connectivity indices) -> 
	-> Molecular structure
		-> Chemical properties (pKa, logP, solubility)
		-> Biological properties (activity, toxicity, pharmacokinetic)
- Mol. structure -> representation -> descriptors -> feature selection and mapping -> activities
