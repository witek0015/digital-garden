## Lecture notes:
Data types in R:
	- Numeric - either ints or floats, doubles are floats.
	- Integer - holds only the multiplicity of something and is indicated by suffix -L (ex. 4L, 8L). Allows for showing for example number of samples
	- Character - just regular strings, but the addition does not work, unlike in Python
	- Logical - TRUE, FALSE - we can check statements:
		- d = c(1, 2, 3, 4)
		- d1 = d > 2
		- d1 = FALSE, FALSE, TRUE, TRUE
Data storing in R:
	- vectors
	- list
	- matrix
	- dataframe
	- factors
Commands:
	- class(x): returns the class of data
	- typeof(x): returns the type of data (more accurate, shows doubles)
	- head(x): shows us exact number
	- as.{datatype}(x): converts x into given data type
